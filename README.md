A tiny little clone of the fantastic [minutecryptic.com](https://www.minutecryptic.com/) designed for creating and sharing simple cryptics with friends and family.

You can create a new puzzle by visiting the [website](https://danprince.github.io/mini-cryptic/), opening the JavaScript console, and entering the following code:

```js
PUZZLE(
  "The clue that you want to provide",
  "theanswer",
  "Your name"
)
```

Then copy the resulting URL to send that puzzle to someone!
